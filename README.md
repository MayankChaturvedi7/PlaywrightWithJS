# PlaywrightWithJS

Simple Playwright/JS learning repository.

## Contents
- Chapter 1: Basics
  - Chapter 1/01_Basics.js — example `console.log` script

## Quick start
1. Install Node.js (v16+ recommended)
2. Run the example:

```powershell
node "Chapter 1/01_Basics.js"
```

## Notes
- This repo is for Playwright learning; add Playwright tests under a `tests/` folder when ready.
- `.gitignore` is configured to ignore node_modules and Playwright artifacts.
